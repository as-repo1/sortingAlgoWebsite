Bubble Sort

Bubble Sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.

Learn More

Merge Sort

Merge Sort is an efficient, stable, and comparison-based sorting algorithm that divides the unsorted list into smaller sublists, sorts them, and then merges them.

Learn More

TimSort

TimSort is a hybrid sorting algorithm derived from merge sort and insertion sort, designed to perform well on many kinds of real-world data.

Learn More

Heap Sort

Heap Sort is a comparison-based sorting algorithm that uses a binary heap data structure to sort elements.

Learn More

Insertion Sort

Insertion Sort is a simple sorting algorithm that builds the final sorted array one item at a time.

Learn More

Selection Sort

Selection Sort is an in-place comparison sorting algorithm that divides the input list into a sorted and an unsorted region.

Learn More

Tree Sort

Tree Sort is a sorting algorithm that builds a binary search tree from the elements to be sorted and then traverses the tree (in-order) to get the sorted sequence.

Learn More

Shell Sort

Shell Sort is an in-place comparison sort that can be seen as an extension of insertion sort. It reduces the number of comparisons by sorting the elements at different intervals.

Learn More

Bucket Sort

Bucket Sort is a sorting algorithm that divides the input into a number of equally spaced intervals called buckets, and then distributes the elements into the buckets.

Learn More

Radix Sort

Radix Sort is a non-comparative sorting algorithm that sorts data with integer keys by grouping the keys by individual digits.

Learn More

Counting Sort

Counting Sort is an integer sorting algorithm that sorts elements by counting the number of occurrences of each unique element.

Learn More

Cube Sort

Cube Sort is an efficient, in-place sorting algorithm based on the concept of radix sort.
