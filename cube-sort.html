Cube Sort

Cube Sort is a parallel sorting algorithm that distributes the elements into a three-dimensional array and performs multiple rounds of sorting on each dimension.

Algorithm

C++


#include <iostream>
using namespace std;

void cubeSort(int arr[], int n, int m) {
    const int MAX = 1000; // Maximum value in the array
    int count[MAX + 1] = {0};

    for (int i = 0; i < n; i++)
        count[arr[i]]++;

    int index = 0;
    for (int i = 0; i <= MAX; i++) {
        while (count[i]--)
            arr[index++] = i;
    }
}

int main() {
    int arr[] = {5, 3, 8, 4, 2};
    int n = sizeof(arr) / sizeof(arr[0]);
    int m = 1000; // Maximum value in the array

    cubeSort(arr, n, m);

    cout << "Sorted array: ";
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }

    return 0;
}

Python


def cube_sort(arr, m):
    n = len(arr)
    count = [0] * (m + 1)

    for i in range(n):
        count[arr[i]] += 1

    index = 0
    for i in range(m + 1):
        while count[i] > 0:
            arr[index] = i
            index += 1
            count[i] -= 1

arr = [5, 3, 8, 4, 2]
cube_sort(arr, 1000)  # Maximum value in the array

print("Sorted array:", arr)

Explanation

Cube Sort distributes the elements into a three-dimensional array, where each dimension corresponds to a specific range of values. It then performs multiple rounds of sorting on each dimension until the array becomes sorted.

Complexity

The time complexity of Cube Sort is O(n + m), where n is the number of elements in the array and m is the maximum value in the array. The space complexity is O(m) since Cube Sort requires additional space for the counting array.
