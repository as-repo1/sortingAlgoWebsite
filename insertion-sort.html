Sorting Algorithms

Insertion Sort

Insertion Sort is a simple sorting algorithm that builds the final sorted array one item at a time by comparing each item with its previous elements and inserting it into the correct position.

Algorithm

C++


#include <iostream>
using namespace std;

void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;

        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }

        arr[j + 1] = key;
    }
}

int main() {
    int arr[] = {5, 3, 8, 4, 2};
    int n = sizeof(arr) / sizeof(arr[0]);

    insertionSort(arr, n);

    cout << "Sorted array: ";
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }

    return 0;
}

Python


def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1

        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1

        arr[j + 1] = key

arr = [5, 3, 8, 4, 2]
insertion_sort(arr)

print("Sorted array:", arr)

Explanation

Insertion Sort works by repeatedly taking an element from the unsorted part of the array and inserting it into its correct position in the sorted part of the array. It shifts larger elements to the right to make room for the inserted element.

Complexity

The average and worst-case time complexity of Insertion Sort is O(n^2), where n is the number of elements in the array. The best-case time complexity is O(n) when the array is already sorted. The space complexity is O(1) since Insertion Sort sorts the array in-place without requiring additional space.
