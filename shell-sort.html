Shell Sort

Shell Sort is an in-place comparison-based sorting algorithm that starts by sorting elements far apart from each other and progressively reduces the gap between elements to perform a final insertion sort.

Algorithm

C++


#include <iostream>
using namespace std;

void shellSort(int arr[], int n) {
    for (int gap = n / 2; gap > 0; gap /= 2) {
        for (int i = gap; i < n; i++) {
            int temp = arr[i];
            int j;

            for (j = i; j >= gap && arr[j - gap] > temp; j -= gap) {
                arr[j] = arr[j - gap];
            }

            arr[j] = temp;
        }
    }
}

int main() {
    int arr[] = {5, 3, 8, 4, 2};
    int n = sizeof(arr) / sizeof(arr[0]);

    shellSort(arr, n);

    cout << "Sorted array: ";
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }

    return 0;
}

Python


def shell_sort(arr):
    n = len(arr)
    gap = n // 2

    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i

            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap

            arr[j] = temp

        gap //= 2

arr = [5, 3, 8, 4, 2]
shell_sort(arr)

print("Sorted array:", arr)

Explanation

Shell Sort starts by sorting elements far apart from each other and progressively reduces the gap between elements to perform a final insertion sort. The final pass of Shell Sort is equivalent to an insertion sort but applied to a partially sorted array.

Complexity

The time complexity of Shell Sort depends on the chosen gap sequence. The average-case complexity is difficult to determine precisely but is generally better than the worst-case complexity of O(n^2). The space complexity is O(1) since Shell Sort sorts the array in-place without requiring additional space.
