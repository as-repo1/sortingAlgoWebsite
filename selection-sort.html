Sorting Algorithms

Selection Sort

Selection Sort is a simple comparison-based sorting algorithm that divides the input array into a sorted and an unsorted region. It repeatedly selects the smallest (or largest) element from the unsorted region and swaps it with the first element of the unsorted region.

Algorithm

C++


#include <iostream>
using namespace std;

void selectionSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int minIndex = i;

        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex])
                minIndex = j;
        }

        if (minIndex != i)
            swap(arr[i], arr[minIndex]);
    }
}

int main() {
    int arr[] = {5, 3, 8, 4, 2};
    int n = sizeof(arr) / sizeof(arr[0]);

    selectionSort(arr, n);

    cout << "Sorted array: ";
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }

    return 0;
}

Python


def selection_sort(arr):
    n = len(arr)
    for i in range(n - 1):
        min_idx = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]

arr = [5, 3, 8, 4, 2]
selection_sort(arr)

print("Sorted array:", arr)

Explanation

Selection Sort divides the array into a sorted region and an unsorted region. It repeatedly finds the smallest (or largest) element from the unsorted region and swaps it with the first element of the unsorted region, expanding the sorted region.

Complexity

The average, worst-case, and best-case time complexity of Selection Sort is O(n^2), where n is the number of elements in the array. The space complexity is O(1) since Selection Sort sorts the array in-place without requiring additional space.
